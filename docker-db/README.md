### Docker utils
> With docker-compose

##### MySQL
- Start mysql: 
    > - Clone this repository.
    > - Run the following in the terminal (in the repo folder): 
    > - `docker-compose up -d`
    > 
    > - Long version (for one service)
    > 
    > - `docker-compose run -d --service-ports mysql-reviewtel`
